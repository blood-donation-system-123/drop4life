About Drop4Life

Drop4Life is a simple blood donation management system created to help users register as donors, request blood, and access donor information easily. All data is stored locally in your browser for demo purposes.

This project demonstrates how a real blood donation system works – with donor registration, blood request matching, and searchable donor lists.
